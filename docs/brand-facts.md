# Breren — Established Facts

Concrete decisions already in effect, as opposed to the naming philosophy
in [naming.md](naming.md).

## Status

Breren is the operating company; it is not just a proposed name.

## Domain & subdomain scheme

Each product gets its own subdomain under `breren.com`:

- `<product>.breren.com` — the product's web app
- `api.<product>.breren.com` — the product's backend API
- `<purpose>@<product>.breren.com` — email addresses (e.g. privacy,
  support, transactional sender)

## Platform/package namespace

Android artifacts use the `com.breren` package root for company-owned
packages.
